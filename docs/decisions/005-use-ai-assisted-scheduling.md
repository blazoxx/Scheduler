# ADR-005: Use AI-Assisted Scheduling

## Status
Accepted

## Date
YYYY-MM-DD

## Context

Users naturally describe appointments using conversational language.

## Decision

Use AI to convert natural language into structured scheduling constraints while keeping booking validation deterministic.

## Alternatives Considered

- Rule-based parser
- Fully AI-driven booking

## Consequences

### Advantages

- Better user experience
- Flexible interpretation
- Easy prompt improvements

### Trade-offs

- AI latency
- Output validation required

## Future Considerations

Support follow-up conversations and context-aware scheduling.
